# open-inspector JSON output - schema v1

Generated from `pkg/model.Module` by `make jsonschema`; do not edit by hand.

This documents the object carried under the `module` key of the `--json` envelope. The envelope wraps it with three scalar fields: `schema_version` (int), `tool` (string), and `version` (string).

## Module (root)

Module is the root inspection result for a single Terraform/OpenTofu module directory.

| Field | Type | Required | Description |
